## Utilité de la commande `npm search @types/bcryptjs`

La commande suivante permet de rechercher le package de types TypeScript pour `bcryptjs` sur le registre npm :

```bash
npm search @types/bcryptjs
```

### À quoi ça sert ?

- **Vérifier l’existence** du package `@types/bcryptjs`
- **Consulter les informations** : version, description, popularité
- **Confirmer l’installabilité** pour bénéficier de l’autocomplétion et de la vérification de types lors de l’utilisation de `bcryptjs` en TypeScript

> Utiliser ce package améliore l’expérience de développement avec TypeScript en apportant une meilleure sécurité et productivité.
